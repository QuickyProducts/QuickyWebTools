What does this tool actually do?

An SL animation is a list of "joints" (skeleton bones) with keyframes per joint. This tool takes every joint that appears in the hand file and uses it to replace the corresponding joint from the body file. Joints that only exist in one of the two files are left untouched.

The loop alignment trick: hand animations are usually their own clean loops (start and end poses match). When packed raw into a longer container animation, the loop point falls in the middle of the hand cycle — and it stutters on every repeat. The "align to loop window" option shifts the hand keyframes so their full cycle sits exactly between loop_in and loop_out of the body animation. A hold keyframe is also inserted at t=0 so the hands don't drift during the ease-in.

If the hand file isn't a clean loop (e.g. a gesture snippet), uncheck the box.
