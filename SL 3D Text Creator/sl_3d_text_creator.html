📖 SL Upload Guide

Single .dae: Quick test upload. SL will auto-decimate LODs (often poor for text).

SL Bundle .zip: Extract and use the bundled LOD .dae files in the SL mesh uploader's per-LOD slots. Result: text reads great from far away with low Land Impact.

Per-face slots: The mesh has 3 face IDs — fill (letter front+back), sides (extruded edge), accent (flower pollen etc.). After upload, edit the object → Texture tab → select face → apply different materials.

PBR: Either copy baseColor / metallic / roughness values from the active preset into SL's material editor, or import the bundled material_preset.gltf as a Material asset and apply it.

Land Impact estimate: Rough heuristic based on tri count + LOD count. Real LI depends on the SL upload & sim — always check there.
