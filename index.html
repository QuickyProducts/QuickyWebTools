Quicky Tools for Second Life

SL Tools

Browser-based utilities for Second Life content creation. Everything runs locally - no uploads, no install, no account. Each tool is a single self-contained HTML file you can also download and use offline.

AC

Animation Combiner

merge any two .anim

Merge two SL animations into one - per-bone priority decides which file wins, and the result loops cleanly. Ideal for adding a hand pose to a dance.

Launch
AP

Animation Priority Changer

set priority on .anim files

Batch-change the priority inside SL .anim files - live 3D skeleton preview, bit-perfect export.

Launch
SS

Sprite Sheet Maker

gif → sprite sheet

Slice animated GIFs into a single sprite-sheet texture, ready to drive texture animations in SL - smart cropping and palette optimisation included.

Launch
TX

3D Text Creator

text → 3D mesh

Type a word, pick a font and depth, export a clean Collada .dae mesh with live 3D preview, triangle count and bundled LODs.

Launch
HD

HUD Buttons Creator

buttons grid → mesh

Lay out a grid of HUD buttons, pick shapes and colours, export an optimised low-poly mesh whose face indices stay aligned with the button order.

Launch
AM

Alpha Maker

body alpha mask painter

Paint over the body parts you want to hide on a UV template and download the alpha mask PNG - brush, rect and ellipse tools, undo, custom templates.

Launch
MS

Music Slicer

songs → ≤30s SL clips

Slice songs into loudness-normalised ≤30s WAV clips that beat SL's sound-clip limit - waveform trimming, metadata titles, one-click ZIP export.
