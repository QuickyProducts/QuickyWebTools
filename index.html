★ SL TOOLS ★

Neon Routine for Second Life Creators

Browser-based utilities for Second Life content creation. Everything runs locally in your browser — no uploads, no install, no account. Drop a file in, get a file out.

AC

Animation Combiner

body + hands → combo

Merge a body animation with a separate hands animation into a single BVH file. Useful for stacking gestures onto poses without overwriting finger curls.

Launch
AP

Animation Priority Changer

set priority on .anim files

Rewrite the priority byte inside a SL internal-format .anim file without re-uploading. Bump a pose from priority 2 to 4 in seconds.

Launch
SS

Sprite Sheet Maker

gif → sprite sheet

Slice an animated GIF, video, or image sequence into a single power-of-two sprite-sheet texture, ready to drive an animated face in SL.

Launch
TX

3D Text Creator

text → 3D mesh

Type a word, pick a font and depth, export a clean DAE/Collada mesh ready to upload as 3D text in Second Life.

Launch
HD

HUD Buttons Creator

buttons grid → mesh

Lay out a grid of HUD buttons, generate the mesh and matching texture in one go. The face indices stay aligned with the button order.
